## Getting started

1. clone or fork this repository

   ```sh
   git clone https://github.com/Gutierrez-16/Sistema_Riga.git
   ```

2. install dependencies

   ```bash
   npm install
   ```

3. run the project
   ```bash
   npm run dev
   ```
4. Open your browser and visit

   [http://localhost:5173 üòé](http://localhost:5173)
